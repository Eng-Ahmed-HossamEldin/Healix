This package contains the Healix Flutter project with login/auth screens and the latest implemented screens:
- Workout Library
- Water Intake
- Sleep Tracking
- Stress Logging
- Human Coach
- Daily Tips & Motivation
- Analytics Dashboard
- Weekly Report
- Progress Comparison

Run:
flutter clean
flutter pub get
flutter run

If your local Flutter version asks to repair platform scaffolding, run:
flutter create .
flutter pub get
flutter run
